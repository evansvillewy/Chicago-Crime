Chicago Crime Data API
Data Source for Crime Analysis

This dataset reflects reported incidents of crime that occurred in the City of Chicago. Data is extracted from the Chicago Police Department's CLEAR (Citizen Law Enforcement Analysis and Reporting) system. In order to protect the privacy of crime victims, addresses are shown at the block level only and specific locations are not identified.

Machine Learning and Modeling

sklearn.ensemble.RandomForestClassifier
Classifier

A random forest is a meta estimator that fits a number of decision tree classifiers on various sub-samples of the dataset and uses averaging to improve the predictive accuracy and control over-fitting.

RandomForestClassifier
sklearn.svm.LinearSVC
Linear Classifier alternative to SVC

Similar to SVC with parameter kernel=’linear’, but implemented in terms of liblinear rather than libsvm, so it has more flexibility in the choice of penalties and loss functions and should scale better to large numbers of samples.

Linear Support Vector Classification
sklearn.linear_model.SGDClassifier
Stochastic Gradient Descent

A simple yet very efficient approach to discriminative learning of linear classifiers under convex loss functions such as (linear) Support Vector Machines and Logistic Regression.
